# 🕸️ Algoritmos e Teoria dos Grafos 

Este repositório reúne implementações de conceitos de Matemática Discreta focados em Teoria dos Grafos. O objetivo dos códigos é realizar a modelagem de problemas complexos, cálculo de distâncias entre pontos específicos e renderização visual de redes estruturadas.

### 🎯 Por que isso importa?
A modelagem através de grafos é a estrutura fundamental por trás de algoritmos modernos de Inteligência Artificial e Ciência de Dados, sendo a base para otimização de rotas, sistemas de recomendação e análise de redes sociais.

### 💻 Tecnologias e Bibliotecas Utilizadas
- **Python:** Linguagem base para a estruturação lógica.
- **NetworkX:** Criação, manipulação e estudo da estrutura e dinâmica de redes complexas.
- **Matplotlib:** Geração das visualizações matemáticas e renderização dos nós e arestas.

### 📂 Arquivos Principais
- `resolucao_grafo_rede.ipynb`: Algoritmo para identificar os caminhos ótimos e distâncias entre formas geométricas específicas (triângulos, quadrados, círculos) sem sobreposição de arestas.
- `manipulacao_grafos_t2.ipynb`: Scripts complementares para geração e manipulação de topologias em grafos.
